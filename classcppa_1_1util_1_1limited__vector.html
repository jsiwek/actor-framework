Detailed Description

template<typename T, size_t MaxSize>
class cppa::util::limited_vector< T, MaxSize >

A vector with a fixed maximum size (uses an array internally).

Warning
This implementation is highly optimized for arithmetic types and does not call constructors or destructors properly.
