Detailed Description

Provides a platform independent type name and a (very primitive) kind of reflection in combination with object.

The platform independent name is equal to the "in-sourcecode-name" with a few exceptions:

  • std::string is named @str
  • std::u16string is named @u16str
  • std::u32string is named @u32str
  • integers are named @(i|u)$size
    e.g.: @i32 is a 32 bit signed integer; @u16 is a 16 bit unsigned integer
  • the anonymous namespace is named @_
    e.g.: namespace { class foo { }; } is mapped to @_::foo
