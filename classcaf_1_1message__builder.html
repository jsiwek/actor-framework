Detailed Description

Provides a convenient interface to create a message from a series of values using the member function append.

Member Function Documentation

message caf::message_builder::to_message ( )

Converts the internal buffer to an actual message object.

It is worth mentioning that a call to to_message does neither invalidate the message_builder instance nor clears the internal buffer. However, calling any non-const member function afterwards can cause the message_builder to detach its data, i.e., copy it if there is more than one reference to it.
