Detailed Description

A buffer implementation with configurable final size that also supports dynamic growing if needed.

Examples:
announce_1.cpp.

Constructor & Destructor Documentation

cppa::util::buffer::buffer ( size_t  chunk_size,
size_t  max_buffer_size 
)

Creates a buffer holding at most max_buffer_size bytes that allocates memory in steps of chunk_size bytes.

Note
The default chunk_size used by other constuctors is 512 bytes.
The default maximum buffer size is 16MB.
