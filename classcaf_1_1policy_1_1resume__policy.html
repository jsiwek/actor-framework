Detailed Description

The resume_policy concept class.

Please note that this class is not implemented. It only explains the all required member function and their behavior for any resume policy.

Resumes the actor by reading a new message and then calling self->invoke(msg).

This process is repeated until either no message is left in the actor's mailbox or the actor finishes execution.
